## التعليمات التقنية

### رفع المشروع إلى GitHub
تم توفير سكربت `push_to_github.sh` لتسهيل عملية الرفع. لتشغيله، تأكد من وجود صلاحيات التنفيذ ثم قم بتشغيله من الطرفية (Terminal):

```bash
sh push_to_github.sh
```
